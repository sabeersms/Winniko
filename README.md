# Winniko

Winniko is a Flutter-based mobile application for organizing and participating in sports prediction competitions.

## Features

- **Create Competitions**: Host your own private or public tournaments.
- **Predict Scores**: Predict match outcomes for major leagues (Premier League, World Cup, IPL, etc.) and custom tournaments.
- **Live Leaderboards**: Track your points and rank against friends in real-time.
- **Chat**: Discuss matches and predictions with other participants.
- **Notifications**: Get alerted for match starts and results.

## Getting Started

### Prerequisites

- Flutter SDK (latest stable)
- Firebase Project configured (Android/iOS/Web)

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/winniko.git
    cd winniko
    ```

2.  **Install dependencies:**
    ```bash
    flutter pub get
    ```

3.  **Run the app:**
    ```bash
    flutter run
    ```

### Building for Release

To generate a release APK:

```bash
flutter build apk --release
```
